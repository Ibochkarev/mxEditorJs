<strong>⚡ mxEditorJs — блочный редактор Editor.js для MODX 3</strong>

Интегрирует <a href="https://editorjs.io/">Editor.js</a> в административную панель MODX 3 в качестве замены стандартного Rich Text Editor. Контент хранится в каноническом JSON-формате Editor.js, на фронтенде автоматически генерируется HTML-снимок.

<strong>🪟 Блочный редактор</strong>
<ul>
  <li>13 типов блоков: параграф, заголовок, список, чеклист, цитата, таблица, код, raw HTML, embed, изображение, вложение, разделитель, предупреждение</li>
  <li>Inline-инструменты: marker, inline-code, underline, link</li>
  <li>Выравнивание текста (tune) для параграфов, заголовков, списков, цитат</li>
  <li>Undo/Redo — поддержка отмены действий</li>
</ul>

<strong>📺 Embed — встраиваемый контент</strong>
<ul>
  <li>YouTube, RuTube, Vimeo, Twitch, Instagram, Facebook, Twitter/X и другие сервисы</li>
  <li>Вставка по URL через Paste API — достаточно скопировать ссылку и вставить в редактор</li>
  <li>Корректный рендер на фронтенде — блоки отображаются как <code>iframe</code> с поддержкой <code>allowfullscreen</code> и <code>loading="lazy"</code></li>
  <li>Возможность добавить кастомные сервисы через конфигурацию</li>
</ul>

<strong>✨ Ключевые возможности</strong>
<ul>
  <li>Поддержка TV-полей — редактор работает в основном контенте и в дополнительных полях (textarea + richtext)</li>
  <li>Загрузка медиа — drag-and-drop изображений и файлов через MODX Media Sources</li>
  <li>Браузер файлов — встроенный просмотр с навигацией по директориям</li>
  <li>Автодополнение ссылок — поиск ресурсов MODX при вставке ссылок</li>
  <li>Миграция HTML → Editor.js — автоматическая конвертация существующего HTML-контента</li>
  <li>Профили инструментов — предустановленные (default, minimal, blog, full) и пользовательские</li>
  <li>Полноэкранный режим и режим Source Preview (просмотр сгенерированного HTML)</li>
  <li>Локализация — русский и английский, наследование локали менеджера</li>
  <li>CSS-пресеты — настраиваемые классы для изображений и ссылок</li>
</ul>

<strong>🚀 Быстрый старт</strong>
<ol>
  <li>Установите пакет через Extras → Installer</li>
  <li>В системных настройках установите <code>which_editor</code> = <code>mxEditorJs</code></li>
  <li>Откройте любой ресурс — в поле контента появится Editor.js</li>
</ol>

<strong>📋 Требования</strong>
<ul>
  <li>MODX Revolution 3.0.3+</li>
  <li>PHP 8.2+</li>
  <li>Node.js 18+ (только для сборки из исходников)</li>
</ul>
