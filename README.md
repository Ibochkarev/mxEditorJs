# mxEditorJs — блочный редактор Editor.js для MODX 3

Редактор контента для MODX 3 на базе [Editor.js](https://editorjs.io/). Работает вместо стандартного TinyMCE/CKEditor: вы создаёте контент блоками (заголовок, текст, картинка, видео), а на сайте всё отображается корректно.

## Возможности

- **Блочный редактор** — 13 типов блоков: параграф, заголовок, список, чеклист, цитата, таблица, код, raw HTML, embed, изображение, вложение, разделитель, предупреждение
- **Поддержка TV** — редактор работает в основном контенте ресурса и в дополнительных полях (Template Variables) типа `textarea` с опцией richtext
- **Загрузка медиа** — drag-and-drop загрузка изображений и файлов через MODX Media Sources
- **Браузер файлов** — встроенный просмотр файлов с навигацией по директориям
- **Автодополнение ссылок** — поиск ресурсов MODX при вставке ссылок
- **Миграция HTML → Editor.js** — автоматическая конвертация существующего HTML-контента
- **Профили инструментов** — предустановленные наборы (default, minimal, blog, full) и пользовательские
- **Полноэкранный режим** и **режим Source Preview**
- **Локализация** — русский и английский языки, наследование локали менеджера
- **CSS-пресеты** — настраиваемые CSS-классы для изображений и ссылок
- **Undo/Redo** — поддержка отмены действий
- **Выравнивание текста** — блочный tune для параграфов, заголовков, списков, цитат

## Установка

### Через Package Manager

1. Скачайте `mxeditorjs-*.transport.zip`
2. В менеджере MODX: **Пакеты → Установщик → Загрузить пакет**
3. Установите пакет

### Из исходников (разработка)

```bash
# Клонируйте в директорию Extras
cd /path/to/modx/Extras/
git clone <repo-url> mxEditorJs
cd mxEditorJs

# Установите зависимости и соберите фронтенд
npm install
npm run build

# Соберите транспортный пакет
php _build/build.php
```

## Быстрый старт (3 шага)

1. **Система → Системные настройки** → найдите `which_editor` → выберите **mxEditorJs**
2. Убедитесь, что `mxeditorjs.enabled` = **Да**
3. Откройте любой ресурс — в поле контента появится блочный редактор

## Настройка

Все настройки находятся в пространстве имён `mxeditorjs` в системных настройках MODX. Подробное описание — в [docs/CONFIGURATION.md](docs/CONFIGURATION.md).

| Настройка | По умолчанию | Описание |
|---|---|---|
| `mxeditorjs.enabled` | `true` | Включить/выключить редактор |
| `mxeditorjs.profile` | `default` | Активный профиль инструментов |
| `mxeditorjs.enabled_tools` | _(пусто)_ | Переопределение профиля: список инструментов через запятую |
| `mxeditorjs.image_mediasource` | `1` | ID Media Source для изображений |
