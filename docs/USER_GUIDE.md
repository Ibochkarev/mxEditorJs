## Что такое mxEditorJs

mxEditorJs — редактор контента для MODX 3. Вместо одного большого поля вы работаете с **блоками**: параграф, заголовок, картинка, видео, таблица и т.д. Каждый блок можно редактировать, перемещать или удалять отдельно.

**Плюсы:**
- Простой интерфейс — меньше путаницы
- Чистый HTML на сайте — без лишнего кода
- Удобно добавлять видео — просто вставьте ссылку

---

## Начало работы

### Как включить редактор

1. **Система → Системные настройки**
2. Найдите настройку **which_editor** → выберите **mxEditorJs**
3. Убедитесь, что **mxeditorjs.enabled** = **Да** (в пространстве имён mxeditorjs)
4. Откройте любой ресурс — в поле контента появится блочный редактор

> Если редактор не появился — см. раздел [Редактор не появляется](#редактор-не-появляется) в конце документа.

## Работа с редактором

### Добавить новый блок

- **Enter** в конце блока — новый параграф
- Нажмите **«+»** слева от пустого блока — откроется меню, выберите нужный тип (заголовок, список, картинка и т.д.)

### Переместить блок

Наведите на блок → слева появится иконка ⋮⋮ → перетащите в нужное место.

### Удалить блок

Кликните ⋮⋮ слева от блока → **Delete**.

### Форматирование текста

Выделите текст — появится панель:
- **B** — жирный
- **I** — курсив
- **Link** — ссылка
- **Code** — код
- **Marker** — выделение цветом
- **U** — подчёркивание

## Типы блоков

### Параграф

Основной текстовый блок. Поддерживает inline-форматирование и выравнивание (лево, центр, право).

### Заголовок (Header)

Заголовки уровней H1–H6. Для смены уровня нажмите на иконку заголовка в тулбаре блока.

### Список (List)

Маркированный или нумерованный список. Переключение типа — через настройки блока.

### Чеклист (Checklist)

Список с чекбоксами. Каждый пункт можно отметить галочкой.

### Цитата (Quote)

Блок цитаты с необязательной подписью автора.

### Таблица (Table)

Редактируемая таблица. Добавление строк и столбцов через контекстное меню. Первая строка может быть заголовком.

### Код (Code)

Блок для вставки программного кода. Содержимое не форматируется.

### Raw HTML

Блок для вставки произвольного HTML-кода. Используйте с осторожностью — код вставляется без фильтрации.

### Разделитель (Delimiter)

Горизонтальная линия для визуального разделения контента.

### Предупреждение (Warning)

Блок с заголовком и текстом предупреждения. Используется для важных заметок.

### Вложение (Attaches)

Загрузка файла для скачивания. Поддерживает: PDF, DOC, DOCX, XLS, XLSX, PPT, PPTX, TXT, CSV, ZIP, RAR, 7Z.

## Изображения

### Загрузка

1. Нажмите **«+»** → **Image** (или **Изображение**)
2. Выберите способ:
   - **Загрузить** — перетащите файл или выберите из файловой системы
   - **Обзор** — откроется встроенный браузер Media Source

### Браузер изображений

Встроенный браузер позволяет:
- Просматривать файлы в директориях Media Source
- Навигировать по папкам (кнопки **Назад** и **В корень**)
- Выбрать существующее изображение кликом

### Настройки изображения

После вставки изображения доступны настройки:
- **Подпись** (caption) — текст под изображением
- **Рамка** (border) — добавить рамку
- **На всю ширину** (stretch) — растянуть по ширине
- **Фон** (background) — добавить фон
- **CSS-стиль** — выбор из предустановленных классов (настраивается администратором)

### Поддерживаемые форматы

JPG, JPEG, PNG, GIF, WebP, SVG. Максимальный размер файла — 5 МБ (настраивается через `mxeditorjs.max_upload_size`).

## Ссылки

### Вставка ссылки

1. Выделите текст
2. Нажмите иконку **Link** в inline-тулбаре
3. Введите URL или начните набирать название ресурса MODX — появится автодополнение

### Автодополнение ресурсов

При вводе текста в поле URL редактор ищет ресурсы MODX по:
- Заголовку страницы (pagetitle)
- Расширенному заголовку (longtitle)
- ID ресурса

### Дополнительные настройки ссылки

- **Target** — открытие в том же окне / новом окне / во фрейме
- **Rel** — nofollow, noopener, noreferrer, sponsored, ugc
- **CSS-класс** — выбор из предустановленных (кнопка, внешняя ссылка и др.)

## Видео и встраиваемый контент

### Как добавить видео

1. Скопируйте ссылку на видео (например, из YouTube или RuTube)
2. В редакторе создайте пустой блок (Enter) или нажмите «+»
3. **Вставьте ссылку** (Ctrl+V / Cmd+V) — блок появится автоматически

**Важно:** Кнопки «Embed» в меню нет. Вставляйте ссылку напрямую — редактор сам распознает сервис.

### Поддерживаемые сервисы

YouTube, RuTube, Vimeo, Twitch, Instagram, Twitter/X, CodePen, Facebook и др.

### Как это выглядит на сайте

После сохранения ресурса видео отображается на сайте как встроенный плеер (iframe). Пользователи смогут смотреть видео прямо на странице.

## Дополнительные поля (TV)

mxEditorJs работает в TV-полях (Template Variables) типа **Textarea** с включённой опцией **Rich Text**.

### Настройка TV для Editor.js

1. Создайте TV с типом ввода **Textarea**
2. В настройках TV установите опцию **Rich Text** = `Да`
3. Назначьте TV нужному шаблону
4. Откройте ресурс → вкладка **Дополнительные поля** — в TV появится Editor.js

Каждое TV-поле получает собственный экземпляр редактора с отдельным хранилищем данных.

## Полноэкранный режим и просмотр HTML

- **⛶ Fullscreen** — развернуть редактор на весь экран
- **</> Source** — посмотреть, как контент будет выглядеть в HTML (только просмотр)

## Миграция из HTML

Если в ресурсе уже есть HTML (от TinyMCE, CKEditor и т.п.), при первом открытии появится окно миграции:

1. Просмотрите превью — сколько блоков будет создано
2. Нажмите **Мигрировать** — HTML превратится в блоки
3. Или **Отмена** — начнёте с пустого редактора

Миграция распознаёт заголовки, параграфы, списки, цитаты, картинки, таблицы и код.

## Горячие клавиши

| Клавиша | Действие |
|---------|----------|
| `Enter` | Новый блок |
| `Backspace` в пустом блоке | Удалить блок |
| `Tab` | Переместить блок в список (вложенность) |
| `Shift+Tab` | Вынести из вложенности |
| `Ctrl+B` / `Cmd+B` | Жирный |
| `Ctrl+I` / `Cmd+I` | Курсив |
| `Ctrl+K` / `Cmd+K` | Вставить ссылку |
| `Ctrl+Z` / `Cmd+Z` | Отменить |
| `Ctrl+Shift+Z` / `Cmd+Shift+Z` | Повторить |

## Частые вопросы

### Редактор не появляется

1. **Система → Системные настройки** → `which_editor` = **mxEditorJs**
2. В настройках mxeditorjs: `mxeditorjs.enabled` = **Да**
3. `use_editor` = **Да**
4. **Система → Очистить кэш**

### Редактор не появляется в дополнительных полях (TV)

1. TV должен быть типа **Textarea**
2. Опция **Rich Text** = **Да**
3. Откройте вкладку **Дополнительные поля** — редактор загрузится при появлении поля

### Не добавляется видео

Кнопки «Embed» нет. **Скопируйте ссылку** на видео (YouTube, RuTube и т.д.) и **вставьте** (Ctrl+V) в пустой блок — блок создастся сам.

### Картинка не загружается

Обратитесь к администратору. Нужно проверить: Media Source, права на папку, размер файла (до 5 МБ), формат (JPG, PNG, GIF, WebP, SVG).
