# Превью для mxEditorJs

Материалы для публикации дополнения **mxEditorJs** на [modstore.pro](https://modstore.pro) и для превью в GitHub/соцсетях.

## Файлы

| Файл | Назначение |
|------|------------|
| **DESCRIPTION.md** | Короткое описание дополнения для summary/анонса |
| **modstore-description.md** | Полное HTML-описание для карточки ModStore |
| **logo-variant-3.png** | Текущий PNG-вариант логотипа для карточки пакета и превью |
| **banner-1200x630.png** | Баннер для соцсетей, Open Graph и GitHub preview |

## Использование

- **ModStore:** используйте `DESCRIPTION.md` как short description и `modstore-description.md` как полное описание.
- **Обложка пакета:** при необходимости подготовьте отдельный экспорт логотипа нужного размера на основе `logo-variant-3.png`.
- **Соцсети/GitHub:** `banner-1200x630.png` можно использовать как `og:image` и preview banner.
- Если понадобится SVG или другой размер, сделайте новый экспорт из исходного макета.

Папка `_preview/` не входит в транспортный пакет установки.
