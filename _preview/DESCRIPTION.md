# mxEditorJs — блочный редактор Editor.js для MODX 3

Интеграция Editor.js в административную панель MODX 3 как замена Rich Text Editor. Контент хранится в JSON, на фронтенде рендерится HTML.

- 13 типов блоков: параграф, заголовок, список, чеклист, цитата, таблица, код, raw HTML, **embed** (YouTube, RuTube, Vimeo и др.), изображение, вложение, разделитель, предупреждение
- Поддержка TV-полей типа textarea с richtext
- Загрузка медиа через Media Sources, браузер файлов, автодополнение ссылок
- Миграция HTML → Editor.js, профили инструментов (default, minimal, blog, full)
- Полноэкранный режим, Source Preview, локализация (ru/en), выравнивание текста

Требования: MODX 3.0.3+, PHP 8.2+.
